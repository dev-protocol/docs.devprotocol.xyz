Admin pages on Clubs allows plugins to use default style sheets to provide a standardized console UX for users.

import './plugin-admin-styles.css'

### Colors

The UI colors of the admin pages will automatically switch between light/dark mode depending on the user's preferred color scheme.

The following words can be used as values for Tailwind's utilities such as **`bg-`**, **`border-`**, and **`text-`**.

| Token         | Light mode                                                                  | Dark mode                                                                        | Main uses                  | Example          |
| ------------- | --------------------------------------------------------------------------- | -------------------------------------------------------------------------------- | -------------------------- | ---------------- |
| `surface-200` | <div class="i" style={{ background: 'var(--hs-theme-surface-200)' }}></div> | <div class="i" style={{ background: 'var(--hs-theme-surface-dark-200)' }}></div> | Backgrounds                | bg-surface-200   |
| `surface-300` | <div class="i" style={{ background: 'var(--hs-theme-surface-300)' }}></div> | <div class="i" style={{ background: 'var(--hs-theme-surface-dark-300)' }}></div> | Backgrounds                | bg-surface-300   |
| `surface-400` | <div class="i" style={{ background: 'var(--hs-theme-surface-400)' }}></div> | <div class="i" style={{ background: 'var(--hs-theme-surface-dark-400)' }}></div> | Backgrounds                | bg-surface-400   |
| `surface-600` | <div class="i" style={{ background: 'var(--hs-theme-surface-600)' }}></div> | <div class="i" style={{ background: 'var(--hs-theme-surface-dark-600)' }}></div> | Backgrounds                | bg-surface-600   |
| `surface-ink` | <div class="i" style={{ background: 'var(--hs-theme-surface-ink)' }}></div> | <div class="i" style={{ background: 'var(--hs-theme-surface-dark-ink)' }}></div> | Texts/borders on `surface` | text-surface-ink |
| `error-200`   | <div class="i" style={{ background: 'var(--hs-theme-error-200)' }}></div>   | <div class="i" style={{ background: 'var(--hs-theme-error-dark-200)' }}></div>   | Backgrounds                | bg-error-200     |
| `error-300`   | <div class="i" style={{ background: 'var(--hs-theme-error-300)' }}></div>   | <div class="i" style={{ background: 'var(--hs-theme-error-dark-300)' }}></div>   | Backgrounds                | bg-error-300     |
| `error-400`   | <div class="i" style={{ background: 'var(--hs-theme-error-400)' }}></div>   | <div class="i" style={{ background: 'var(--hs-theme-error-dark-400)' }}></div>   | Backgrounds                | bg-error-400     |
| `error-600`   | <div class="i" style={{ background: 'var(--hs-theme-error-600)' }}></div>   | <div class="i" style={{ background: 'var(--hs-theme-error-dark-600)' }}></div>   | Backgrounds                | bg-error-600     |
| `error-ink`   | <div class="i" style={{ background: 'var(--hs-theme-error-ink)' }}></div>   | <div class="i" style={{ background: 'var(--hs-theme-error-dark-ink)' }}></div>   | Texts/borders on `error`   | text-error-ink   |
